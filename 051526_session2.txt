Model Invariants (Don't Break)
mountAssignments is authoritative for what's at each mount. ShipDef.turrets[i].weapon is only used by autoSeedMountAssignments() on ship-equip.
Slot keys: "turret_<i>" / "gun_<i>" / "engine_<i>" matching the JSON array index.
Type filter (HardpointKind.accepts):
turret slot ↔ category turret/turrets
gun slot ↔ category gun/guns
engine slot ↔ category thruster/steering/engine/engines
Thrust/turn count only from mounted engines (PlayerProfile.totalEngineThrust iterates mountAssignments not installedOutfits).
Engine emitters in ShipNode.installEngineEmitters use mountAssignments["engine_<i>"], not installedOutfits.
Beam routing in GameScene.tickBeamWeapon consults mountAssignments[slot] only; no JSON default fallback.
Save file format uses explicit CodingKeys (snake_case) — no keyEncodingStrategy (would mangle dict keys like installed_outfits entries).
Legacy save migration: PlayerProfile.loadFromSave rewrites human-name keys ("Heavy Laser Turret") to snake_case IDs, then auto-seeds mount assignments if the save's mount_assignments is empty.
Recent Architectural Decisions
Joystick uses InputState.targetHeading: Float? (absolute world bearing). OfflineSim slews via min(turnRate*dt, |diff|) — no overshoot possible regardless of turn speed.
Heat scale: maxHeat = mass × 60, dissipation rate = mass × heat_dissipation/sec. ES convention.
Thrust acceleration: accel = totalThrust × 4000 / totalMass. TURN_TO_RADPS = 50.
Beam classifier: weapon.reload >= 1.0 ⇒ beam (continuous instant-hit). Else projectile firing every reload seconds, per-shot damage = stat × reload.
Asteroid colliders are alpha-traced from the baked texture; the parent body rotates (not the sprite) so collider stays aligned with the visible mesh.
Beam targeting rules: asteroid must be selectedBody; hostile-faction ships (none yet) are auto-hit; player's own ship and planets/suns are skipped (no physics body on planets/suns).
Damage routing: weapon's per-second stats × dt for beams; × reload for per-shot projectiles. Goes through OfflineSim.applyDamage.
Known Loose Ends (Not in This Turn's Scope)
